# Level 5: Exam Simulation - CKA Practice Tests

Full CKA exam simulations with time limits and scoring.

## Exam Format

- **Duration**: 2 hours
- **Questions**: 15-20 tasks
- **Passing Score**: 66% (74% for CKA)
- **Environment**: Real Kubernetes cluster

## Practice Exams

**Note**: Full-length practice exams are now located in the `practice-exams/` directory at the root level. This folder contains exam simulation exercises as part of the progressive learning path.

### Exam Simulation Exercises

### Time: 2 hours | Questions: 18

1. **Troubleshooting** (30 min)
   - Fix pod in CrashLoopBackOff
   - Fix service with no endpoints
   - Fix deployment rollout stuck
   - Fix node NotReady
   - Fix PVC not binding

2. **Deployment** (20 min)
   - Create deployment with 5 replicas
   - Configure health probes
   - Set resource limits
   - Update image
   - Rollback if needed

3. **Networking** (25 min)
   - Create ClusterIP service
   - Create NodePort service
   - Configure Ingress
   - Create NetworkPolicy
   - Test connectivity

4. **Storage** (15 min)
   - Create PV and PVC
   - Use in StatefulSet
   - Verify persistence

5. **Security** (20 min)
   - Create ServiceAccount
   - Configure RBAC
   - Apply Pod Security Standards

6. **Maintenance** (10 min)
   - Backup resources
   - Drain a node
   - Check cluster health

## Scoring Guide

- **90-100%**: Excellent! Ready for exam
- **80-89%**: Good, minor review needed
- **70-79%**: Needs more practice
- **Below 70%**: Focus on fundamentals

## Exam Day Tips

1. Read all questions first
2. Prioritize by difficulty
3. Time box each question
4. Verify your work
5. Use kubectl explain
6. Don't panic if stuck
